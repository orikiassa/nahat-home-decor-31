
המשתמש שואל אם אפשר שבכל דף מוצר יהיו כמה תמונות (גלריה). כרגע דף המוצר (`src/routes/product.$handle.tsx`) מציג רק תמונה אחת — הראשונה. אבל ה-GraphQL query כבר מביא עד 5 תמונות (`images(first: 5)` ב-`PRODUCT_BY_HANDLE_QUERY`), אז הדאטה כבר זמין — צריך רק לבנות UI של גלריה.

## התוכנית: גלריית תמונות בדף המוצר

### מה ייבנה
גלריית תמונות עם תמונה ראשית גדולה + תמונות ממוזערות (thumbnails) מתחת. לחיצה על תמונה ממוזערת מחליפה את התמונה הראשית.

### פריסה
```text
┌─────────────────┐  ┌──────────────────┐
│                 │  │  שם המוצר        │
│  תמונה ראשית    │  │  ₪מחיר            │
│   (גדולה)       │  │  תיאור            │
│                 │  │  [הוסיפו לסל]    │
└─────────────────┘  └──────────────────┘
┌──┐ ┌──┐ ┌──┐ ┌──┐
│t1│ │t2│ │t3│ │t4│  ← thumbnails
└──┘ └──┘ └──┘ └──┘
```

### שינויים בקוד
**קובץ יחיד:** `src/routes/product.$handle.tsx`
- הוספת `useState` למעקב אחרי אינדקס התמונה הנבחרת.
- שליפת כל התמונות מ-`product.images.edges` (במקום רק הראשונה).
- רינדור התמונה הנבחרת בגדול + רצועת thumbnails מתחת.
- אם יש רק תמונה אחת — לא להציג את רצועת ה-thumbnails (חוויה נקייה).
- מצב hover/selected על thumbnail עם border בצבע accent.
- במובייל: thumbnails בגלילה אופקית.

### חוויית משתמש
- לחיצה על thumbnail = החלפה מיידית של התמונה הראשית (ללא ניווט).
- ה-thumbnail הנבחר מודגש עם מסגרת.
- שמירה על RTL ועל אותו עיצוב מינימליסטי קיים.

### הערה לגבי Shopify
התמונות נשלפות אוטומטית מהמוצר ב-Shopify Admin — כל תמונה שתעלה למוצר שם תופיע בגלריה (עד 5). אם תרצה יותר מ-5, נוכל להעלות את המגבלה ב-query.

### מה לא משתנה
- אין שינוי בחנות Shopify, אין שינוי ב-API או ב-cart store.
- אין dependencies חדשים.
